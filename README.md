# Portfolio

Personal frontend portfolio built with Next.js.

## Live

https://your-portfolio-link.com

## About

This is a single-page portfolio to present my work, skills, and contact information, with a focus on clean UI and smooth interactions.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Features

- Responsive layout
- Animated components
- Project showcase section
- Tech stack carousel
- Contact links and CV download

## Approach

The project focuses on simplicity and visual quality, with attention to spacing, typography, and subtle animations to improve user experience without adding unnecessary complexity. Next.js was used as a structured foundation for the project, providing built-in routing, optimized builds, and a better development experience compared to a plain React setup.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Structure

- pages/index.tsx — main page
- components/ — UI components
- public/ — images, icons, and assets
